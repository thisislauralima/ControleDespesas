
# Controle de Despesas

Aplicação Front-End para trabalho de extensão.
Por ela podemos cadastrar despesas, seus valores e categorias de maneira simples.


## Operações

#### Obter despesas

Ao carregar a tela inicial da aplicação as despesas cadastradas serão exibidas numa lista.


#### Cadastrar despesa

Cadastro de despesa incluindo: nome, valor e categoria.


#### Editar despesa

Edição de despesa já existente. Valores que podem ser modificados: nome, valor e categoria.

Para editar basta selecionar a despesa desejada na lista.

#### Excluir despesa
Excluir despesa existente.

Basta selecionar a despesa e em seguida apertar em excluir.
